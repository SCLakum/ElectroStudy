Area of Square

Problem Statement

Given a side of a square, your task is to calculate its area.

Input:

The first line of the input contains the side of the square.

Output: You just have to print the area of a square

Algorithm:

  1. Read the input integer representing the length of the square's side.
  2. Calculate the area of the square by multiplying the side length by itself.
  3. Output the result as a single integer.

Example:

Sample Input:
3

Sample Output:
9

Sample Input:
6

Sample Output:
36

Java Code   
 
import java.io.*;
import java.util.*;

class Main {

    public static int Square_Area(int Side){
      int Area = Side*Side;
      return Area;
    }

    public static void main (String[] args) {
      Scanner SC = new Scanner(System.in);
      int Side = SC.nextInt();
      System.out.print(Square_Area(Side));
    }
}
